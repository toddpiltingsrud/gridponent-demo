A boolean value. If true, a "required" attribute is added to the input and a span.required is added after the input. The default CSS sets the content of the span to a bootstrap exclamation sign.

commands

Row-level commands can be created in two ways:

  1. A simple comma-separated string: { commands: 'Edit,Delete' }
  2. An array of objects: 
    { commands: [
    {text: 'Edit', value:'edit'}, 
    {text: 'Delete', value:'delete'},
    {text: 'Custom', func:function( dataItem ) { // do something to the row's data object }, glyphicon: 'glyphicon-asterisk'}
] }

Command objects can have the following properties:

  • text: The text displayed to the user.
  • value: The name of the command. Predefined values that are automatically handled by gridponent include 'edit' and 'delete'.
  • btnClass: The name of a CSS class to add to the button element.
  • glyphicon: One of the Bootstrap glyphicon classes for adding an icon to the left of the button text.
  • func: A function for a custom command. If the 'value' property is a string representing a function path, gridponent resolves it and sets the 'func' property to the resolved function.
